# Trig Function Graph (Node.js Web Server Edition)

HTML5 Canvas を使用して、単位円の回転運動とそれに同期するサイン波（正弦波）・コサイン波（余弦波）をリアルタイムに描画するインタラクティブなアニメーション作品です。
Node.js を用いた軽量な Web サーバー上で動作します。

## 💡 特徴・改良点

- **滑らかなアニメーション**: `setInterval` から `requestAnimationFrame` へ変更し、ブラウザの描画レートに同期したスムーズな描画を実現。
- **バグの修正**: Canvas のパス描画（`beginPath`）を最適化し、円と座標軸の描画が干渉して塗りつぶされてしまう問題を解決。
- **可視性の向上**: 円の中心位置やグラフの開始点を調整し、数学的な波形のつながり（単位円からどのように波が紡ぎ出されるか）をより直感的に理解しやすいデザインに変更。
- **モダンな記述**: 変数宣言（`let`/`const`）や全体のレイアウトを現代的な JavaScript / CSS にリファクタリング。

## 🛠️ 動作環境

- **Node.js**: v14 以上の環境を推奨

## 🚀 使い方

### 1. ファイルの配置
プロジェクトフォルダ内に以下の2つのファイルを配置します。

- `server.js` (Node.js サーバー用コード)
- `index.html` (フロントエンド描画用コード)

### 2. サーバーの起動
ターミナル（またはコマンドプロンプト）を開き、対象のフォルダへ移動して以下のコマンドを実行します。

```bash
node server.js
